# Keyboard Focus Order per Screen

Define tab order and focus entry points for each screen.

## Global

- First focusable element: Skip to content link
- Second: Global search
- Third: Primary nav
- Modal focus trap: first field in modal

## Dashboard

- Page header action buttons
- StatCards (left to right)
- Charts (left to right)
- Recent activity table

## Reservations List

- FilterBar fields left to right
- Primary action (New Reservation)
- DataTable header (sortable columns)
- DataTable rows
- Pagination controls

## Reservation Detail

- Header actions (Check-in, Modify)
- Summary cards
- Tabs
- Active tab content fields
- Folio panel actions

## New Reservation Wizard

- Stepper current step
- Form fields
- Next/Back actions
- Summary panel

## Check-in Wizard

- Stepper
- Room selection table
- Deposit fields
- Confirm button

## Check-out Wizard

- Folio lines
- Payment fields
- Confirm button

## Room Calendar

- Date range selector
- Calendar grid (row then column order)

## Housekeeping Board

- FilterBar
- Column headers
- Task cards within column

## Room Status

- FilterBar
- Room cards in numeric order

## Reports

- Report list
- Filters
- Charts
- Table

## Users and Roles

- Search/filter
- DataTable rows
- Drawer form fields

## ABAC Policies

- DataTable rows
- Drawer JSON editor

## Settings

- Tabs
- Table rows
- Add button
- Form fields
