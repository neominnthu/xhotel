# Frontend Design System

## Visual Direction

- Clean, modern, and warm neutral palette
- High contrast for operational clarity

## Typography

- Primary font: "Space Grotesk", sans-serif
- Secondary font: "Source Serif 4", serif (headings only)
- Use clear hierarchy: H1 28px, H2 22px, body 14-16px

## Color Tokens

- --bg: #f7f5f1
- --surface: #ffffff
- --text: #1a1a1a
- --primary: #1e6f5c
- --accent: #f2b705
- --danger: #c0392b

## Layout Rules

- Top nav + left sidebar across all pages
- Consistent page headers with actions on the right
- Grid and card layout for main content

## Spacing and Grid

- Spacing scale: 4, 8, 12, 16, 24, 32
- Max content width: 1280px
- Use 12-column grid for dashboards and reports

## Components

- Button: primary, secondary, danger
- Status badges: pending, confirmed, checked-in, checked-out, canceled
- Table: sticky header, row actions
- Modal: consistent sizes and footer actions
- Form: labeled inputs, inline validation

## States and Feedback

- Buttons: hover, active, disabled states defined
- Tables: loading skeletons and empty states
- Forms: inline errors and summary banner

## Pages (Core)

- Dashboard
- Reservations list and calendar
- Reservation detail (folio and guest)
- Check-in/out wizard
- Housekeeping board
- Reports

## Page Templates

- List pages: filters + table + pagination
- Detail pages: header + summary + tabs
- Wizard pages: stepper + validation per step

## Accessibility

- Keyboard navigation for key flows
- Clear focus states
- Minimum 4.5:1 contrast

## Responsive Breakpoints

- Mobile: 360-599px
- Tablet: 600-1023px
- Desktop: 1024px and up
