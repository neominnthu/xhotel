# UI Interaction Specs for Keyboard-First Workflows

This document defines keyboard interactions to speed up front desk operations.

## Global Shortcuts

- / : focus global search
- g then r: go to reservations list
- g then c: go to check-in wizard
- g then o: go to check-out wizard
- ? : open shortcuts help

## Global Search

- Type to search guest name, phone, reservation code
- Enter opens top result
- Esc closes search

## Tables

- Arrow keys move row focus
- Enter opens row detail
- Space toggles row selection
- Ctrl+E opens edit

## Forms

- Enter moves to next field
- Shift+Enter moves to previous field
- Ctrl+S saves form
- Esc cancels and closes modal

## Wizard Steps

- Alt+N next step
- Alt+P previous step
- Alt+C confirm

## Reservation List

- F filter focuses filter bar
- N creates new reservation
- C cancels selected reservation

## Check-in

- A assigns selected room
- D opens deposit modal
- C confirms check-in

## Check-out

- P opens payment modal
- C confirms check-out

## Housekeeping Board

- Arrow keys navigate cards
- M marks task complete

## Accessibility

- Visible focus indicator for all interactive elements
- Skip-to-content link at top of page
